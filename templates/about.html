Precise Point Positioning Service

To use this service:

  • The only thing to do is Uploading a RINEX observation file
  • PLEASE SET YOUR RINEX NAME IN THIS FORMAT : "yourstationname"MMDD.yyo
  • The program will automatically download:
    • 1-Precise orbits and ephemerides for the observation day and next day
    • 2-Satellite clock corrections

The system processes your data to deliver centimeter-level accuracy positioning results.

This product is designed and developed by Python and (HTML + CSS + JS).

The method used to provide precise coordinates is Precise Point Positioning (PPP).
